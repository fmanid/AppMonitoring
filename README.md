### App Monitoring
Base Laravel 6.*
<hr>

How To Install : <br>
- Composer install <br>
- Duplicate .env.example to .env <br>
- php artisan key:generate <br>
- php artisan migrate --seed <br>
- php artisan serve <br>
- Go To Login Page http://localhost:8000/login
